# Quantum Compiler Infrastructure V2

This project implements a multi-stage compilation pipeline for quantum circuits targeting LLVM-based backends.

## Core Architecture
- Lexer: Custom rust-based tokenizer for QASM strings.
- Optimization: Gate-folding algorithms to reduce decoherence risk.
- Backend: Direct mapping to WebAssembly for browser-based simulation.

## Technical Constraints
The system must maintain Zero Abstraction overhead during the transpilation process to ensure micro-architectural timing precision is preserved across heterogeneous quantum processor units (QPUs).
