# System Overview

* Next.js 14 App Router platform with a hybrid Server/Client architecture.
* Prisma v6 with PostgreSQL for indexing metadata and tracking entities.
* MDX is the primary content source of truth, parsed and rendered progressively.
* High-fidelity custom UI strictly preserving Framer Motion animations (e.g., Starfield/Oscilloscope hero canvas).
* Strict separation between Public Frontend and Premium Admin Dashboard.
* **Content Architecture**: Whitepaper Series, Reading Paths (difficulty-based categorization), and Giscus integration for peer-reviewed discourse.
