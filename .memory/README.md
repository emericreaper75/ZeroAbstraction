# ZeroAbstraction AI Memory System

This directory contains the operational engineering memory for autonomous AI agents working on ZeroAbstraction. 
It ensures architectural consistency, tracks decisions, and maintains system context across sessions.

## Principles
- **Lightweight**: Uses markdown and structured JSON.
- **Incremental**: Only update affected files, no large rewrites.
- **Separation of Concerns**: Architecture, errors, patterns, and state are isolated.
- **Priority**: P0 (Architecture/Security) > P1 (Rules) > P2 (Patterns) > P3 (Session notes)
