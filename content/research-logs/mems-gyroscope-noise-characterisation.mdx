## Context

This log records the noise characterisation procedure applied to a MEMS gyroscope (datasheet: $\text{ARW} = 0.01 \, °/\sqrt{\text{hr}}$, $\text{BI} = 1 \, °/\text{hr}$). Raw angular rate data was collected at $f_s = 400 \, \text{Hz}$ over a 4-hour static period.

## Allan Variance

For a time series $\Omega(t)$ of angular rates, the **Allan Variance** is defined as:

$$
\sigma_A^2(\tau) = \frac{1}{2} \left\langle \left[ \bar{\Omega}(t + \tau) - \bar{\Omega}(t) \right]^2 \right\rangle
$$

where $\bar{\Omega}(\tau)$ is the average angular rate over an interval $\tau$. The computation proceeds via overlapping cluster averages from $N$ samples at rate $f_s$:

$$
\sigma_A^2(\tau) = \frac{1}{2\tau^2(N - 2m)} \sum_{j=1}^{N-2m} \left[ \theta_{j+2m} - 2\theta_{j+m} + \theta_j \right]^2
$$

where $m = \tau f_s$ and $\theta_j = \sum_{k=1}^{j} \Omega_k / f_s$ is the cumulative angle.

## Noise Parameter Extraction

From the Allan deviation $\sigma_A(\tau)$ vs. $\tau$ log-log plot, noise parameters are read at characteristic slopes:

| Noise Type | Slope | Read Point | Extracted Value |
|---|---|---|---|
| Angle Random Walk (ARW) | $-1/2$ | $\tau = 1 \, \text{s}$ | $0.0092 \; °/\sqrt{\text{hr}}$ |
| Bias Instability (BI) | $0$ | minimum | $0.87 \; °/\text{hr}$ |
| Rate Random Walk (RRW) | $+1/2$ | $\tau = 3 \, \text{hr}$ | $0.12 \; °/\text{hr}^{3/2}$ |

Both ARW and BI match the datasheet within measurement uncertainty. The RRW was not specified by the manufacturer.

## Data Collection and Processing

```python
import numpy as np
import matplotlib.pyplot as plt

def allan_variance(omega: np.ndarray, fs: float) -> tuple[np.ndarray, np.ndarray]:
    """
    Compute overlapping Allan variance for a gyroscope time series.
    
    Args:
        omega: Angular rate samples [rad/s or deg/s]
        fs:    Sampling rate [Hz]
    
    Returns:
        tau:   Averaging times [s]
        avar:  Allan variance [units^2]
    """
    N = len(omega)
    # Cumulative angle
    theta = np.cumsum(omega) / fs
    
    max_m = N // 4  # Maximum cluster size
    m_vals = np.unique(np.logspace(0, np.log10(max_m), 200).astype(int))
    
    tau = m_vals / fs
    avar = np.zeros(len(m_vals))
    
    for i, m in enumerate(m_vals):
        # Overlapping cluster differences
        diffs = theta[2*m:] - 2*theta[m:-m] + theta[:-2*m]
        avar[i] = np.sum(diffs**2) / (2 * (m/fs)**2 * len(diffs))
    
    return tau, avar

# Load raw gyro data
data = np.load('gyro_static_4h.npy')  # shape: (5_760_000, 3) [deg/s]
omega_z = data[:, 2]  # Z-axis

tau, avar = allan_variance(omega_z, fs=400.0)
adev = np.sqrt(avar)  # Allan deviation

# ARW: sigma * sqrt(tau) at tau=1s (read from slope=-1/2 region)
tau_1 = 1.0
arw_idx = np.argmin(np.abs(tau - tau_1))
arw = adev[arw_idx] * np.sqrt(3600)  # Convert to deg/sqrt(hr)
print(f"ARW = {arw:.4f} deg/sqrt(hr)")

# Bias instability: minimum of ADEV
bi_idx = np.argmin(adev)
bi = adev[bi_idx] * 3600  # Convert to deg/hr
print(f"Bias instability = {bi:.3f} deg/hr")
print(f"  at tau = {tau[bi_idx]:.1f} s")
```

## Observations

The angle random walk of $0.0092 \, °/\sqrt{\text{hr}}$ places this device in the tactical-grade category (ARW $< 0.1 \, °/\sqrt{\text{hr}}$), consistent with its intended application in UAV attitude estimation.

The bias instability floor occurs at $\tau \approx 320 \, \text{s}$. Beyond this point, the Allan deviation increases due to rate random walk — likely from thermal drift. A temperature-compensation model should reduce this to below $0.05 \, °/\text{hr}$.

### Temperature Sensitivity

A secondary test measured bias drift as a function of temperature over $-10°\text{C}$ to $+60°\text{C}$:

$$
\hat{b}(T) = b_0 + \alpha (T - T_0) + \beta (T - T_0)^2
$$

Fitted parameters: $\alpha = 0.023 \, °/\text{hr}/°\text{C}$, $\beta = 8.4 \times 10^{-5} \, °/\text{hr}/°\text{C}^2$. Applying this model reduces bias instability by approximately $60\%$.
